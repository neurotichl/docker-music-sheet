# Docker Music Sheet
Run flask app in docker conatainer

# Run in Docker
```
docker pull neurot1chl/music-sheet-pdf
docker run -d -p 5000:5000 neurot1chl/music-sheet-pdf
```

# Run from folder
```
python run.py
```
